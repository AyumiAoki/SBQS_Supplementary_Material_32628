# Supplementary Material

This repository contains the supplementary material for the article:

**“The AI-Native Manifesto: Where Are Brazilian Companies?”**

Submitted to the Brazilian Symposium on Software Quality (SBQS 2026).

This study investigates the extent to which software development practices reported by professionals from Brazilian companies align with the AI-Native Manifesto. The study examines development processes, team roles, artifacts, quality assurance, agent use, and reuse strategies through a qualitative survey based on semi-structured interviews.

### 📁 File Descriptions

#### 📌 Consent Form — TCLE.pdf

Contains the informed consent form presented to the participants. The document describes the study objectives, research procedures, voluntary participation, anonymity guarantees, and use of the collected data.

#### 📄 P1.pdf–P11.pdf

Contain the anonymized transcripts of the semi-structured interviews. Each file corresponds to one participant and preserves the participant identifiers used in the article.

#### 📊 Qualitative Analysis.pdf

Contains the qualitative analysis conducted in the study, including interview excerpts, codes, themes, and their relationships with the six principles of the AI-Native Manifesto. The file supports the traceability between participants’ reports, the reported findings, and the analytical structure used in the article.

#### 📌 Initial Interview Script.pdf

Contains the initial version of the semi-structured interview guide and checklist used in the pilot interview.

#### 📌 Final Interview Script.pdf

Contains the revised interview guide and checklist used during the main data collection. The researchers refined this version after the pilot interview to improve the clarity of the questions.

#### 📄 README.md

Provides an overview of the supplementary material and describes the purpose of each file.

### 🔒 Anonymization

The material identifies participants as **P1–P11** and companies as **C1–C10**. The researchers removed personal, organizational, and project information that could reveal participants’ identities.
